# UI UX Pro Max Design Intelligence

When working on UI/UX tasks, follow these guidelines from the UI UX Pro Max skill.

## Design System Generation Process

1. **Identify Product Type** - Determine the industry (SaaS, Fintech, Healthcare, E-commerce, Wellness, etc.)
2. **Select UI Style** - Choose from 67 available styles based on product type
3. **Generate Color Palette** - Apply industry-appropriate colors
4. **Select Typography** - Choose font pairings that match the mood
5. **Apply Best Practices** - Follow accessibility and UX guidelines

## Quick Reference

### Recommended Styles by Industry

| Industry | Styles |
|----------|--------|
| SaaS | Minimalism, Glassmorphism, Soft UI, Bento Grid |
| Fintech | Glassmorphism, Dark Mode, Trust Blue palette |
| Healthcare | Accessible Design, Inclusive Design, Clean Medical |
| E-commerce | 3D Preview, Hero-Centric, Conversion-Optimized |
| Wellness | Neumorphism, Organic Biophilic, Soft colors |
| AI Products | AI-Native UI, Zero Interface, Glassmorphism |

### Pre-Delivery Checklist

Always verify before delivering UI:

- [ ] SVG icons (Heroicons/Lucide), no emojis
- [ ] cursor-pointer on clickable elements
- [ ] Hover states with 150-300ms transitions
- [ ] WCAG AA contrast (4.5:1 minimum)
- [ ] Visible focus states for keyboard nav
- [ ] prefers-reduced-motion support
- [ ] Responsive: 375px, 768px, 1024px, 1440px

### Anti-Patterns to Avoid

- Emojis as icons
- Missing cursor-pointer
- Harsh/jarring animations
- Poor color contrast
- Non-responsive layouts
- Missing focus states
