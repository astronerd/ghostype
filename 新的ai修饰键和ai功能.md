# 功能描述 

- 现在修饰键的作用不够好，翻译和随心记是好功能但可定制性和扩展性不够，所以我们要重新设计修饰键功能 
**新的功能我想叫 Ghost Morph**

首先修饰键的激活将会简化，我们依然按住快捷键说话 松手上屏或进入润色 
当用户在按住 option 的时候按下修饰键，即可进入对应的模式
用户刚进去的时候 只有两个修饰键 一个是 shift 一个是 command 还有一个是 call ghost twin 
shift 是随心记
command 是 ghost command

command 就是用户可以说：帮我写一段 nano banana 提示词，我要生成一个XXX要求是什么什么
我们就会直接调用 AI 生成提示词上屏。这意味着其实，每个功能，都有自己独立的skill

call ghost twin 就是用户说 燕峰喊我去吃饭，帮我拒绝他
ghost twin 就会以用户的口吻和语言习惯发一个话拒绝张燕峰

ghost twin 的选项要标注当前 ghost twin 的等级 比如（Ghost Twin Lv1）

用户支持删除这些修饰键 或编辑这些修饰键
比如把 shift 改成 CAP lock 
理论上，修饰键可以被设置成任何按键（但是要提示用户不要和系统的快捷键和应用快捷键冲突）

用户亦可以增加修饰键 去完成更多功能：我们自带的有翻译功能，用户可以设置要翻译成什么语言

这里就涉及到 修饰键 的功能 我们管他叫 什么呢？你可以想想 可以叫 power 或者 skill，skill 的卡片可以编辑，比如翻译就可以编辑翻译的语言。command 不能设置 也有卡片 卡片上是介绍，记笔记也可以设置是润色后记录笔记 还是直接记录笔记。（之所以每一个技能都有一个卡片是因为，要支持用户自己设置技能，自己设置的技能是可编辑的）

好，现在说技能的行为，行为分四种，一种是直接输出文字，这是当光标在屏幕内且没有选中任何文字的时候。一种是对文字进行改写，这是当用户选中文字且支持输入的时候。一种对文字进行解释，这是当用户选中文字但不能输入的时候。最后是不能输入且没有选中的状态。以上四种。一二，用户说的话处理后上屏。三四，要有一个悬浮卡片，把用户说的话 调用的 skill 以及输出显示在卡片上，并且支持复制和分享。
